# Task Tracker REST API

A production-ready Task Tracker REST API built with Node.js 20, TypeScript, Express 4, Jest, Zod, and a clean layered architecture.

## Tech Stack

- Node.js 20
- TypeScript
- Express 4
- Zod
- Jest
- Supertest
- Helmet
- CORS
- Morgan
- dotenv

## AI Sub-Agents

This project was built using four defined AI sub-agents.

### 1. Architecture Agent

Responsible for designing the project structure, TypeScript setup, and layered backend architecture.

### 2. Backend API Agent

Responsible for implementing REST routes, controllers, services, repositories, and CRUD logic.

### 3. Validation & Quality Agent

Responsible for adding Zod validation, custom errors, validation middleware, not-found middleware, and global error handling.

### 4. Testing & Documentation Agent

Responsible for creating Jest/Supertest tests, README documentation, API examples, and project usage instructions.
